Project Overview

Link-It is a LinkTree clone, that allows users to create personalized landing pages where they can manage all their online links and social media pages. It offers user authentication, user profile functionality, allowing users to create and manage their profiles(profile images are also supported). It also includes a forgotten password feature that allows users to easily reset their old password and regain access to their account. HTML, SCSS, JavaScript, Django, PostgreSQL
